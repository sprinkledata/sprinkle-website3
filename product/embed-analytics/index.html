Key features

Super-fast Embedding with Secure Access Control

Integrate Sprinkle's powerful analytics directly into your applications with simplified embedding and robust access control.

Simplified Embedding

Generate a unique embed link for your dashboards within the Sprinkle settings. This link acts as the access key.

Then, you can easily embed this link directly within your application framework for a seamless user experience.

Interactive Visualizations

Empower executives and decision-makers with access to consolidated reports, performance summaries, and interactive charts directly within their existing business intelligence tools.

Role-Based Data Control

When configuring the embed link, you can specify an API key and secret associated with a particular user role within your organisation.

The data users see in the embedded dashboard is then dynamically shown based on the permissions linked to that role.

Questions

Frequently asked questions

Can’t find the answer you’re looking for? Feel free to contact us

How do I embed a dashboard in my application?

It's easy! Locate the "Embed" option for your dashboard. Generate a link and choose an API key and secret to set data visibility.

This link acts as the access key. You can then embed this link directly within your application framework. Read more here.

Do users need to log in to see embedded dashboards?

No. User credentials aren't required to view embedded dashboards. The data access is controlled by the API key and secret embedded within the link itself.

Can I share the embed link with anyone?

Yes, but with caution! The level of data users see in the embedded dashboard depends on the API key and secret associated with the link. Ensure you only share the link with relevant users.
